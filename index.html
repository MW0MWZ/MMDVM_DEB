🚀 Quick Start

Select your Debian/Ubuntu version:
Debian 12 (Bookworm) / Ubuntu 22.04-24.04 / Raspberry Pi OS
Debian 11 (Bullseye) / Ubuntu 20.04 / Raspberry Pi OS Legacy
Debian 13 (Trixie) / Testing
1
Add the GPG key to your system:
wget -qO - https://deb.pistar.uk/hamradio.gpg | sudo gpg --dearmor -o /usr/share/keyrings/hamradio.gpg
2
Add the repository to your sources:
echo "deb [signed-by=/usr/share/keyrings/hamradio.gpg] https://deb.pistar.uk/ bookworm main" | sudo tee /etc/apt/sources.list.d/hamradio.list
3
Update and install packages:
sudo apt update && sudo apt install mmdvmhost dmrclients dstarclients dstarrepeater ysfclients nxdnclients p25clients pocsagclients fmclients aprsclients

📦 Core Software

mmdvmhost
The main MMDVM Host software for digital voice modes. Supports DMR, D-Star, YSF, P25, NXDN, POCSAG, and FM. Includes calibration tools and OLED display support on ARM platforms.
MMDVMHost MMDVMCal RemoteCommand
dstarrepeater
Dedicated D-Star repeater controller with support for multiple hardware interfaces including DVAP, DV-RPTR, GMSK modems, and sound cards. Features GPIO control for PTT on ARM platforms, voice announcements, and beacon support.
dstarrepeaterd dstarrepeaterconfig

🔄 Gateway & Client Packages

dmrclients
DMR ecosystem with gateway and cross-mode converters. Routes between multiple DMR networks (Brandmeister, DMR+, TGIF) and converts to other digital modes.
DMRGateway DMR2YSF DMR2NXDN
dstarclients
Comprehensive D-Star gateway system with IRC DDB connectivity, D-Star Gateway, remote control, STARnet server, and various support tools.
ircDDBGateway DStarGateway remotecontrold starnetserverd More tools
ysfclients
Complete YSF/Fusion ecosystem including gateway, parrot test server, DG-ID routing, and cross-mode converters to all major digital voice protocols.
YSFGateway YSFParrot DGIdGateway YSF2DMR YSF2NXDN YSF2P25
nxdnclients
NXDN digital voice support with gateway for reflector connections, parrot test server, and cross-mode converter to DMR networks.
NXDNGateway NXDNParrot NXDN2DMR
p25clients
P25 Phase 1 and Phase 2 support with gateway for reflector connections and parrot test server for signal testing.
P25Gateway P25Parrot
pocsagclients
DAPNET (Decentralized Amateur Paging Network) gateway for POCSAG paging. Connects to the DAPNET network for amateur radio pager messaging.
DAPNETGateway
fmclients
FM Gateway provides analog-to-digital bridging, allowing FM repeaters to connect with digital voice networks for cross-mode operation.
FMGateway
aprsclients
APRS Gateway bridges between APRS-IS (Internet Service) and RF networks, enabling position reporting and messaging across both mediums.
APRSGateway
